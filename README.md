# CodeAlpha Python Internship Tasks

Python programming projects completed as part of the CodeAlpha internship program.

## Tasks Completed

### Task 1 — Hangman Game
A text-based Hangman game where the player guesses a hidden word letter by letter with a maximum of 6 wrong guesses.
- **Concepts:** `random`, loops, lists, string methods

### Task 2 — Stock Portfolio Tracker
A stock tracker that calculates total investment based on user input and hardcoded stock prices. Saves results to a `.txt` file.
- **Concepts:** dictionaries, arithmetic, file handling, loops

### Task 3 — Web Scraper Automation
Automatically scrapes and saves webpage titles from multiple URLs using `requests` and `re`.
- **Concepts:** `requests`, `re`, file handling, exception handling

### Task 4 — Rule-Based Chatbot
A conversational chatbot that responds to user input using predefined rules and keyword matching.
- **Concepts:** functions, if-elif, loops, input/output

## How to Run
1. Clone the repository
2. Navigate to any task folder
3. Run the Python file: `python filename.py`
